## 🛠 Installation & Set Up

1. 패키지 설치

   ```sh
   npm i
   ```

2. .env 파일 생성

   ```sh
   # nodemailer
   AUTH_USER=yourmail
   AUTH_PASS=your token or password
   NODEMAILER_PORT= nodemailer port

   # Google analytics
   G_ID=your ga ID
   ```

3. 로컬 실행

   ```sh
   npm run dev
   ```
